## 게임 방법
- 먼저 Game Start 버튼을 눌러 스테이지 선택화면으로 갑니다.
- 원하는 스테이지를 선택하고 Game Play 버튼을 눌러 스테이지를 진행합니다.
- 플레이어가 죽거나 스테이지 몬스터가 다 죽으면 스테이지가 종료됩니다.
